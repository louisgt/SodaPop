Getting help
============

Having trouble? We'd like to help!

* Try the :doc:`FAQ <faq>` -- it's got answers to some common questions.
* Looking for specific information? Try the :ref:`genindex` or :ref:`modindex`.
* Search for questions on the archives of the `sodapop-users mailing list`_.
* Report bugs with SodaPop in our `issue tracker`_.

.. _sodapop-users mailing list: https://groups.google.com/forum/#!forum/scrapy-users
.. _issue tracker: https://github.com/louisgt/SodaPop/issues
